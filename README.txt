##################################
#   Flappy Bird Arcade Edition   #
##################################

This is source code (.gml) of a FlappyBird (clone) arcade version.
The game is meant to be installed on an arcade cabinet and has a "insert credits" function.
The code is an adaptation of zbox's version that can be found here: 
http://gmc.yoyogames.com/index.php?showtopic=609927
